Sobre Mim

Olá, eu sou o Robson Camargo, analista de dados com sólida base técnica e paixão por transformar dados brutos em insights estratégicos.

Domino ferramentas essenciais como Python para análise de dados, SQL para manipulação de bancos de dados, Power BI para visualizações impactantes e Excel para soluções ágeis, aliados a conhecimentos em estatística aplicada e princípios de design para visualização de dados.

Atualmente em transição de carreira para a área de dados, desenvolvi projetos práticos que demonstram minha capacidade de solucionar problemas reais através da análise de dados - convido você a explorá-los abaixo.

Desvendando a Mega-Sena: Uma análise estatística completa

Utilizando técnicas de análise de dados e visualização, examinei 2864 concursos da Mega-Sena, revelando comportamentos interessantes que desafiam algumas crenças populares e confirmam outras. Aqui, minha ideia não é descobrir uma "fórmula mágica" para ganhar na loteria, mas sim apresentar uma visão baseada em dados sobre como os números se comportam ao longo do tempo.
